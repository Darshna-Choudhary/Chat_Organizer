# Chat Organizer

Chat Organizer is a project designed to help you manage, organize, and search chat conversations efficiently. Built in JavaScript, this tool provides an intuitive way to classify and filter chat messages for easier tracking across platforms or team discussions.

## Features

- **Organize chats** by topic, user, or date.
- **Search functionality** for quick access to conversations.
- **Easy integration** with various chat platforms (coming soon).
- **Simple, user-friendly interface.**

## Getting Started

To get started, clone this repository:

```bash
git clone https://github.com/Darshna-Choudhary/Chat_Organizer.git
cd Chat_Organizer
```

### Installation

Ensure you have Node.js installed.

```bash
npm install
```

### Running the App

```bash
npm start
```

## Usage

After launching, use the interface to import chat logs, organize messages into categories, and utilize search to find past conversations. Custom sorting and filtering tools are included to help streamline chat overview.
